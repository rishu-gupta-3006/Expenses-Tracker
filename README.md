# 💰 Expense Tracker Web App

A clean and responsive **Expense Tracker** built using **HTML, CSS, and JavaScript** to help users manage their income and expenses efficiently.

---

## 📌 Features

* ➕ Add income and expense transactions
* 🗑️ Delete transactions
* 📊 Real-time balance calculation
* 🧾 Transaction history display
* 🔍 Filter transactions by category
* 📅 Date-wise transaction tracking
* 💾 Data stored using Local Storage (no backend needed)
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla JS)** – Functionality & logic
* **Local Storage API** – Data persistence

## ⚙️ How It Works

* Users can add a transaction with:

  * Description
  * Amount
  * Type (Income/Expense)
  * Category
  * Date

* The app:

  * Stores data in browser **localStorage**
  * Updates balance dynamically
  * Displays filtered transaction list
